# Proyectos_de_Python
Practicando ejercicios de Python Alura Latam

Este Repositorio ha sido creado con el objetivo de practicar los ejercicios de Python de cada lección Avanzada en Alura Latam 

1. Monitoreando ventas de Bruno.
   Este Programa recibe el número de ventas de 2 productos: manzana y plátano, en el periodo de 1 mes, y muestra qué producto fué el más vendido.
   ![frutas jpg](https://github.com/user-attachments/assets/84413227-7f13-45cf-8b3d-ecf039cdb0e8)
   
2. Temperatura de los servidores:
   Cómo funciona:
   El programa pide al usuario la temperatura actual.
   Si la temperatura es mayor que 25°C, muestra una alerta.
   Si no, confirma que la temperatura está dentro del rango permitido.

3. Calculando el tiempo total del proyecto:
    Cómo funciona:
    Pide los días de las tres actividades (A, B y C).
    Si alguno es negativo, muestra un mensaje de error.
    Si todos son válidos, calcula y muestra el total del proyecto.
   
4. Calculando el IMC:
   El programa debe recibir el peso y la altura de una persona y mostrar el valor del IMC.
